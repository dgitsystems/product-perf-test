1. Update CustomerUser.csv with the provider account will be used.
2. Update InventoryData.csv with the request body of Inventory we want to order.
3. Update LineItemName.csv with the service name we want to order.
4. Update LineItemOrder.csv with the request body of Service we want to order.
5. Update OwnerId.csv with the ID of the provider we used to order.
6. Update SalesOrderID.csv with the Sales Order we already created, this will be used for Completing the 1st Task of each Sales Order.
7. Update ServiceName.csv with the name of the service we want to order from Inventory.
